ABOUT US

NEW TYPE — 為未來道路而生

NEW TYPE 是一個專注於未來性能車與實驗車款的品牌。我們結合機械工藝、前沿科技與極限性能設計，打造突破傳統想像的車輛。
每一台車款都像是一件移動的機械藝術品， 從外觀線條、車體結構到性能設定，都以「未來道路」為設計核心。
在 NEW TYPE 的世界裡，汽車不只是交通工具，而是代表速度、自由與科技進化的象徵。
